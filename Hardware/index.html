Input Devices

Keyboard

A keyboard is an input device with a layout of keys, used to type text, issue commands, and provide input to a computer. It may have additional keys and features for various tasks or applications.

Mouse

A mouse is an input device with buttons and a scroll wheel that lets users move a cursor on a screen, enabling precise interaction with graphical interfaces.

SSD

SDD (Solid State Drive) is a storage device that stores data using NAND-based flash memory, providing faster data access and improved system performance.

GPU

A GPU (Graphics Processing Unit) is a computer component that enhances graphics and visual performance by efficiently processing and rendering visual data.

(I/O) Devices

Input/Output (I/O) Devices facilitate communication between a computer and its users by handling input and output functions, enabling seamless interaction.

PSU

A Power Supply Unit (PSU) provides electrical power to a computer, ensuring it operates by converting and delivering electricity to its components.

Output Devices

Headphones

Headphones are a hardware device that can be connected to a computer, laptop, smartphone, or other device for private audio listening, without the need for installation.

Monitor

A monitor is a visual display for computer and electronic device output, showing text, graphics, and more, commonly used for tasks like document viewing, gaming, and internet browsing.

Speaker

Speakers connect to computers for sound output, and they vary in compatibility. Computer speakers work with the sound card to produce sound.
